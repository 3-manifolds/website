Further resources

  • Quick Reference Cards — overview of important functions
  • IRC help channel — directly connects you to an online chat room
  • Screencasts — videos explaining how Sage works
  • Exploring Mathematics with Sage by Paul Lutus — overview of using Sage in various areas of applied mathematics
  • Sage Tutorial by Mike O'Sullivan, Ryan Rosenbaum, and David Monarres
  • Introducció al SAGE — mini tutorial in Catalan by Maria Bras Amorós
  • Sage for Newbies — book by T. Kosan
  • FAQ: Frequently Asked Questions
  • Books — a collection of online books explaining mathematics (calculus, linear algebra, number theory, etc.) using Sage
  • Sage and Cython: A Brief Introduction by Marshall Hampton — a brief introduction to Sage focusing on Cython
  • Number Theory and the RSA Public Key Cryptosystem by Minh Van Nguyen — using Sage to study elementary number theory and the RSA public key cryptosystem
  • Linear Programming in Sage by Nathann Cohen — There are many very good solvers around and they are now available in Sage.
  • Sage and Graph Theory by Nathann Cohen — Sage will not solve your graph problems in polynomial time. But everything that is already written, YOU do not have to write it again!
  • Linear Error-Correcting Codes by David Joyner and Robert Miller — introduces some of Sage's functionality in the theory of error-correcting codes
  • Group Theory and Sage: A Primer by Rob Beezer — a compilation of Sage commands useful for a student studying group theory for the first time
  • Planet Sage Blogs — a collection of blogs by Sage users and developers
  • Search — an online search interface indexing Sage documentation, discussion groups, various web resources, blogs and various other places on the web
Understanding Python is very important for using the full power of Sage. Python is the interface and primary programming language of Sage. Here are some external resources:
  • Non-Programmer's Tutorial for Python — from Wikibooks, the open-content textbooks collection, offspring of Wikipedia and probably the easiest introduction of all
  • Python Tutorial by Guido van Rossum — written by the father of Python
  • Dive Into Python — maybe not so dry as the one above and better suited for beginners
  • Python for Software Design — How to Think Like a Computer Scientist by Allen B. Downey
  • Google I/O: Painless Python Part 1 / Part 2 — talk introducing Python
  • Building Skills in Python — suitable for people who are already comfortable with programming
